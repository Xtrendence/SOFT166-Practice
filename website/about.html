What does it do? This is a web-app that allows you to play Tic-Tac-Toe. You can use a Philips Hue smart bulb to determine whose turn it is, and if a game session is currently active.
How do I use it? To play the game, you have to navigate to the "Application" page, which is accessible through the side menu, or by clicking on its card on the home page. Make sure to set the details of your Philips Hue light in the "Settings" page beforehand, otherwise it'll try to use the default settings.
How does it work? Blue represents X, pink represents O. As per the rules of Tic-Tac-Toe, X always goes first, so when the web app is opened, the light bulb is turned on, and its color is set to blue. Each time the user clicks on a square, the color of the light bulb changes based on whose turn it is. When the game is over, the light is turned off. If the user chooses to play again, the light is turned on again and set to blue.
